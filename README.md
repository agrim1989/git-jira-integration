# KAN Project
This project involves creating an MCP server for Jira integration.

## MCP Server Setup
To setup the MCP server, follow these steps:
1. Research and gather documentation on MCP server setup and Jira integration.
2. Identify and procure the necessary hardware and software resources for the MCP server.
3. Develop a detailed configuration and deployment plan.

## MCP Server Code
The MCP server code is located in `mcp_server.py`.
